0) Hva er fordelen med et eksisterende JavaScript bibliotek som jQuery i motsetning til å
skrive en “rå” JavaScript for dynamisk endring av HTML sider?
- Fordelen med et eksisterende JavaScript bibliotek som jQuery i vs. Rå JS er at det blant annet enda er forskjeller på hvordan de forskjellige nettleserne implementerer blant annet «Event Listener Model». JQuery har samlet koder som har vist seg å virke i det aller fleste moderne nettlesere.
1) Les denne artikkelen
http://www.ibm.com/developerworks/web/library/wa­htmlmark/index.html og svar på følgende
spørsmål: Hva er sterke sider til HTML/CSS og hva er sterke sider til Canvas? HTML/CSS sine sterke sider inkluderer blant annet ar de kan lett forstå tekst som inneholder «styling» info. I hvilke
situasjoner er det meningsfult å bruke Canvas?  Canvas kan ha det vanskelig for seg å konstruere konkrete UI menyer og lignende, i motsetning til HTML. Dersom en ønsker å lage en rik bakgrunn til siden sin kan Canvas enkelt tas i bruk. Det går også an å inkludere animasjoner o.l. Canvas yter seg best dersom det tas i bruk sammen med HTML. HTML elementene bør dreie seg om UI som menyer, navigerings linker og lignende. Canvas, tar seg av grafiske elementer, animasjoner og lignende.  For en bra side bør en HTML/canvas hybrid metode for å utvikle siden tas i bruk.

HTML5 Canvas representerer? Forklar.
HTML5 Canvas klarer å presenter bilder og funksjoner på en nettside på en dynamisk måte. Den åpner for nye grafiske muligheter som kan gjennomføres på en nettside. En kan si at canvas kan fungere som «scriptable images»
